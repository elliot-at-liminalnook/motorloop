# Superseded hand-authored module status

> **Document status:** Historical · **Audience:** Historical readers · **Last reviewed:** 2026-07-12 · **Replacement:** [`status-matrix-generated.md`](status-matrix-generated.md)

This page formerly copied proof counts, contract totals, test counts, and ECP5
timing into a hand-maintained table. It drifted after the FOC datapath was
pipelined and is intentionally no longer a status source.

Use:

- [`current-status.md`](current-status.md) for high-level maturity and honest
  boundaries;
- [`status-matrix-generated.md`](status-matrix-generated.md) for the latest
  combined proof/synthesis summary;
- [`formal/proof_report.md`](../formal/proof_report.md) for proof assumptions and
  per-property results;
- [`synth/synth_report.md`](../synth/synth_report.md) for authoritative ECP5 fit
  and timing; and
- `rtl/contracts/` for each block’s maintained interface and behavior.

The old table is recoverable from repository history if an earlier release needs
to be reconstructed. It is not reproduced here because preserving stale numbers
next to current evidence was the source of the ambiguity.
